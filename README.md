# ChronosEmu
======

Chronos is a C# open-source CFlyFF (Chinese FlyFF)

We develop this emulator for fun and for learn about the CFlyff structure.
/!\ Please respect the credits !

Credits for Chronos :

- Eternion (developer)
- Eastrall (developer)

Informations:
-------

Chronos is developed in C# with Visual Studio 2017.

Database : MySQL

Framework : .Net 4.6.2

App type : Console
